# logoCleaner
Herramienta para borrar el fondo de los logos

El MVP tendrá dos estrategias explícitas: global y border.
No habrá modo por defecto en el core.
El usuario elegirá la estrategia desde CLI.
La lógica común vivirá en pipeline.
Cada estrategia vivirá en su propio módulo.
El resultado será un PNG con transparencia.
