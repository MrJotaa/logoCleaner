# logoCleaner
Herramienta para borrar el fondo de los logos

MVP: No habrá modo por defecto en el core. El usuario elegirá explícitamente una estrategia: global o border. Ambas estrategias vivirán en módulos separados. El pipeline común se encargará de cargar, aplicar máscara y exportar. Más adelante añadiremos auto como recomendador, no como magia oculta.
